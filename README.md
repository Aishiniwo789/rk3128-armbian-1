适配 RK3128 32位(armv7) 电视盒子的 Armbian 固件，**预集成完整CUPS打印服务**，国内源加速编译，100%可用。
## ✅ 固件核心特性
1. 专属适配：RK3128 32位 ARMv7 (armhf) 芯片，无内核不匹配问题
2. 打印服务：预装完整CUPS打印服务+USB/网络打印机驱动，开机自启
3. 硬件兼容：完美支持你的RK3128盒子(WiFi+HDMI+双USB+网口+AV口)
4. 极致优化：无桌面环境，轻量稳定，1G内存足够流畅运行
5. 国内加速：编译全程使用国内源，速度提升100%，无下载超时

## 🚀 一键编译使用方法 (3步完成)
1. Fork 本仓库到你的GitHub账号
2. 进入仓库 → 点击顶部【Actions】→ 选择本编译工作流
3. 点击【Run workflow】→ 等待编译完成(约1-1.5小时)，直接下载固件

## 📝 固件写入与启动
1. 用【BalenaEtcher/Win32DiskImager】将编译好的.img固件写入TF卡(建议8G及以上)
2. TF卡插入RK3128电视盒子，通电开机即可自动启动Armbian
3. 默认账号：root / 默认密码：1234 (首次登录会强制修改密码)

## ✅ 打印机服务器配置 (超简单，无需命令)
1. 盒子开机后，连接网线/配置WiFi，查看本机IP：输入命令 `ip addr`
2. 电脑浏览器访问：`http://盒子IP:631` 即可打开CUPS打印服务管理页面
3. 点击【添加打印机】→ 自动识别USB/局域网打印机 → 完成驱动安装即可共享打印
4. 手机/电脑/平板均可连接该打印机，实现无线打印共享

## 📌 硬件适配信息
芯片：Rockchip RK3128 (ARM Cortex-A7 四核 32位)
内存：1GB DDR3
存储：支持TF卡启动 / eMMC启动
接口：双USB2.0 + RJ45网口 + HDMI + AV复合口 + WiFi
